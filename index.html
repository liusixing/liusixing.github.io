創新

發表於 2016-07-25   |     |   閱讀次數

納塔利莫特曼在為哈弗15屆畢業生做演講的時候提到了兩種不太尋常的力量——「缺乏經驗」以及「無知」

她認為「缺乏經驗」可以讓人隨心所欲的創造，因為經驗告訴你的常識，在你頭腦里形成的常規性的認識會妨礙人們的創造力，缺乏經驗意味著沒有這些東西束縛創造力，因此可以創造出不可思議的成就。而「無知」（也許用「過度自信」更容易理解一些）則會給予人們挑戰哪些常人認為不可能實現的事情的勇氣，並可能因此而成功。

價值評估的能力

發表於 2016-07-24   |     |   閱讀次數

自由的人擁有獨自選擇的權利，然而決定這種權利是發揮正面價值還是負面價值的根本在於——價值評估的能力。

舉個例子，當幾件商品擺在你面前的時候，你可以清晰的認識到哪一件商品的價格是最高的，哪一件最不值錢。

這種能力相信幾乎每個人都可以自然的擁有，甚至並不需要刻意的鍛鍊，在人們逛商場時人們往往可以輕鬆的分辨昂貴和廉價的商品。其實確切的說這還並不能稱得上是「價值評估」而是「價格評估」。
